# Ambient Forge V10 — Open Ambient World

Ambient Forge is free because ambient music should be accessible to everyone.

If Ambient Forge helps you create music, learn music theory, teach students, relax, meditate, or discover new ideas, and you would like to support future development, you can use the Buy Me a Coffee link on the project page.

## Changes in V10

- Added About button with the free ambient music statement.
- Added Donate button: https://buymeacoffee.com/liwgdnxqyw
- MIDI export filename now includes BPM, root, mode, preset, DNA and date.
  Example: AmbientForge_58BPM_D_Dorian_NordicDrift_DNA01_20260704.mid
- More overlap for channels 1–3 and bass support.
- Drone, Chord Body and Upper Extensions sustain longer.
- Reduced long empty gaps in harmonic foundation layers.
- Keeps the DAW-agnostic philosophy: Reaper, Vital, Valhalla Supermassive, Ableton, Logic, Cubase, Studio One etc.
