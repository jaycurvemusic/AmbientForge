Ambient Forge Help

WELCOME TO AMBIENT FORGE

Ambient Forge is a MIDI-first ambient composition engine.

Ambient Forge generates MIDI which can be routed into any DAW.

Supported examples:

  • Reaper
  • Ableton Live
  • Cubase
  • Studio One
  • FL Studio
  • Bitwig
  • Dawsome LOVE

Ambient Forge is DAW-independent.

QUICK START (WINDOWS)

  1. Install loopMIDI.
  2. Create a virtual MIDI port named AmbientForge.
  3. Start Ambient Forge.
  4. Press Connect MIDI Out.
  5. Select the AmbientForge port.
  6. Create 6 instrument tracks in your DAW.
  7. Assign MIDI channels 1–6.
  8. Load instruments.
  9. Press Start Forge.

QUICK START (macOS)

  1. Open Audio MIDI Setup.
  2. Enable IAC Driver.
  3. Create an IAC port.
  4. Select the IAC port in Connect MIDI Out.
  5. Route MIDI channels in your DAW.

CHANNEL MAPPING

CH1 = Root Drone
Long evolving foundation.

CH2 = Chord Body
Main harmonic layer.

CH3 = Upper Extensions
Air and harmonic colour.

CH4 = Motion
Movement and evolution.

CH5 = Sparkles
Shimmer, bells and details.

CH6 = Soft Bass Pulse
Low-frequency support.

RECOMMENDED SOUNDS

CH1: Drones, atmospheres, long pads.

CH2: Pads, strings, warm textures.

CH3: Choirs, high pads, harmonic layers.

CH4: Motion textures, soft pulses.

CH5: Bells, glass, shimmer sounds.

CH6: Sub bass, low drones.

DAW ROUTING EXAMPLE

Track 1 → MIDI Channel 1
Track 2 → MIDI Channel 2
Track 3 → MIDI Channel 3
Track 4 → MIDI Channel 4
Track 5 → MIDI Channel 5
Track 6 → MIDI Channel 6

MIDI EXPORT

Export 6-track MIDI creates:

  • CH1 Root Drone
  • CH2 Chord Body
  • CH3 Upper Extensions
  • CH4 Motion
  • CH5 Sparkles
  • CH6 Soft Bass Pulse

The file can be imported into any DAW.

TIPS

  • Use different sounds on all channels.
  • Avoid using the same preset on every channel.
  • Drone + Pad + Extensions + Motion + Sparkles + Bass gives the best results.

PHILOSOPHY

Ambient Forge starts with a drone and gradually builds a harmonic world around it.

The goal is not a traditional song structure.

The goal is:

Atmosphere
Movement
Space
Emotion
